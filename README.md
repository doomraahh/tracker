Мобильное приложение для отслеживания ежедневных привычек. Каждая привычка имеет прогресс от 0 до 100%. Пользователь может выполнять привычки, пропускать день и сбрасывать прогресс, а также просматривать статистику и историю действий.

## Стек

- Expo ~56
- React Native 0.85
- React 19
- Expo Router — навигация (`app/(tabs)`)
- Redux Toolkit — управление состоянием
- react-redux — интеграция с React
- TypeScript — типизация

## Возможности

- Список привычек с названием, прогресс-баром и текущим значением (%)
- Кнопки: **Выполнить** (+10%), **Пропустить** (−10%), **Сбросить** (0%)
- Вкладка статистики: общее число выполнений и история действий

## Требования

- Node.js 18 или новее
- npm (устанавливается вместе с Node.js)
- Для телефона: приложение **Expo Go** (Android / iOS)

## Установка и запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите dev-сервер:

```bash
npm start
```

3. Откройте приложение одним из способов:

| Способ | Команда / действие |
|--------|-------------------|
| Телефон (Expo Go) | Отсканируйте QR-код в терминале |
| Android-эмулятор | `npm run android` или клавиша `a` |
| Браузер | `npm run web` или клавиша `w` |
| iOS (только macOS) | `npm run ios` или клавиша `i` |

Телефон и компьютер должны быть в одной Wi‑Fi сети.
